Little Bombers Returns help

 

Game overview

Little Bombers Returns is a remake of the famous Dyna Blaster/BomberMan games.

You play from one up to four of little bombers trapped in mysterious castle.

These bombers can drop bombs to explode enemies and bricks.

Your purpose is to destroy all monsters on the stages and leave castle. There are 8 stages by 8 levels.

Good luck!

Multiplayer

You may play this game with your friends on shared keyboard.

There are two types of multiplayer game. In the normal mode of the game you and your friends must fight against evil monsters in cooperation.

In the “Battle” you must destroy bomber(s) of your partner(s).

Advanced options

If you have modern 3d video card, you can use 3D acceleration for enhancing quality of game graphics - try to set option “3D Acceleration” of the “Setup” option of the main menu to “ON” . If your video card is outdated and game is too slowly in 3d accelerated mode or works with bags or graphics artifacts, you can set option “3D Acceleration” to “OFF” .

If game hangs in 3d accelerated mode and you can’t reach game menu, you should reinstall game or delete “config.ini” file.

 

Hardware requirements

P-II 300 Mhz or higher
32 MB RAM or higher
Windows 98, 2000, XP with installed DirectX 7 or higher
DirectX-compatible sound card
4 MB DirectX - compatible video card with support mode 640 x 480 with 65536 colors, AGP recommended.

Note 1: in 3d accelerated mode your video card must be Direct3D-compatible, with 8 MB or above video memory on board, with AGP and with supporting textures of 32-bit with alpha-channel; 64 MB or above system memory recomme nded. Some video cards (outdated and rare models) will not be usable in this mode.

Note 2: with no sound card you can play this game typing "sound=0" in "config.ini" file with standard Windows text editor "NotePad" (you must double click file "config.ini" and edit and save changes).
